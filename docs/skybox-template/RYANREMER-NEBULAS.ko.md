# Ryan Remer 네뷸라 스카이박스

SemionTD용 네뷸라 스카이박스 4종은 Ryan Remer의 [Nebulas in Godot](https://ryanremer.itch.io/nebulas-in-godot) 이미지로 만들었습니다.

- 원본 저장소: [RyanRemer/nebulas](https://github.com/RyanRemer/nebulas)
- 코드 및 저장소 라이선스: MIT
- 이미지 사용 안내: 제작자는 itch.io 댓글에서 상업적 사용이 가능하다고 안내합니다.
- 변환 내용: 4096×1024 가로 스트립을 SemionTD의 2048×1024 큐브맵 아틀라스로 변환했습니다. 위·아래 면은 원본 셰이더의 극점 페이드에 맞춰 중앙으로 갈수록 어두워집니다.

등록 ID는 `ryanremer_nebula_blue_amber`, `ryanremer_nebula_violet`, `ryanremer_nebula_frostfire`, `ryanremer_nebula_ember`입니다.

배포할 때는 원본 저장소의 `LICENSE`를 스카이박스 파일과 함께 보관합니다.
